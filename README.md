# 🌸 DailyBloom - Journaling App

A beautiful, soft, feminine journaling application designed for mindful reflection and emotional tracking. Built with React, TypeScript, and Tailwind CSS.

## ✨ Features

- **Landing Page** - Welcoming hero section with feature highlights
- **Authentication Flow** - Login, Register, and Email Verification pages
- **Dashboard** - View all your journal entries in a beautiful grid layout with search functionality
- **Journal Creation** - Write new entries with mood tracking
- **Journal Details** - View, edit, and delete individual entries
- **Mood History** - Visualize your mood patterns over time with statistics
- **Profile Management** - Update your personal information and view journaling stats
- **Mood Tracking** - Track emotions with pastel mood badges:
  - 🌞 Happy (Yellow)
  - 🌸 Inspired (Pink)
  - ☁️ Calm (Blue)
  - 🌙 Reflective (Lavender)
  - 🌧 Tired (Gray)

## 🎨 Design System

- **Color Palette**: Soft pink gradients (#FFF8F5 to #F8C8DC) on white backgrounds
- **Typography**: Dark brown text (#4A3F35) instead of harsh black
- **Spacing**: Generous padding (24-32px) for a calm, breathable layout
- **Borders**: Rounded corners throughout (16-24px radius)
- **Effects**: Smooth hover animations and subtle shadows
